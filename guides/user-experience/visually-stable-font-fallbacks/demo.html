Visually Stable Font Fallbacks

This demo illustrates how to use font-size-adjust to keep text visually stable when the preferred font is unavailable and the browser swaps to a fallback font.

Without font-size-adjust

Notice the significant difference in character size. If the preferred font is loading, swapping causes a disruptive layout shift.

Hover over the text below to simulate a font swap:

The quick brown fox jumps over the lazy dog

With font-size-adjust

By using font-size-adjust, the fallback font is automatically scaled so its lowercase characters match the target size of the preferred font, preventing drastic layout shifts.
