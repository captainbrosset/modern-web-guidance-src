Component-Specific Themes

By scoping light-dark() variables and the color-scheme property directly to a component class rather than the :root, you can force individual instances of that component to render in a specific theme—regardless of the surrounding page's system preference!

System Preference

This card organically inherits the :root color-scheme, toggling automatically between light and dark modes.

Forced Light Mode

This card has color-scheme: light; applied to it directly. Its light-dark() math will mathematically resolve to the light variables forever!
