**Instant navigation via Speculation Rules API:**

Every page (including the homepage) now includes:

```html
<script type="speculationrules">
{
  "prerender": [
    {
      "source": "document",
      "where": {
        "and": [
          { "href_matches": "/*" },
          { "not": { "href_matches": "/logout" } }
        ]
      },
      "eagerness": "moderate"
    }
  ]
}
</script>
```

This tells the browser to **prerender** same-origin pages when the user shows moderate intent to navigate (e.g., hovering over or clicking a link). The pages load fully in a hidden tab before the user navigates, making the transition feel instant. The `/logout` URL is excluded from prerendering to avoid triggering a state-changing action speculatively. Browsers that don't support the Speculation Rules API simply ignore the tag and navigate normally -- no fallback needed.
