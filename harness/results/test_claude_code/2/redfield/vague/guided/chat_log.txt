Here's a summary of what changed:

### What was removed
- **Wrapper `<div class="container">`** — no longer needed; anchor positioning replaces manual `position: relative` containers
- **All imperative JavaScript** — the `mouseover`/`mouseout` event listeners and `display` toggling are gone entirely

### What was added

**Declarative HTML:**
- `interestfor="tooltip"` on the button — declaratively links the trigger to the tooltip on hover/focus interest
- `popover="hint"` on the tooltip `<div>` — uses the Popover API's `hint` type, purpose-built for tooltips

**CSS Anchor Positioning** (replaces manual `position: absolute` + `bottom: 125%` + `transform`):
- `anchor-name: --tooltip-trigger` on the button to establish it as an anchor
- `position-anchor`, `position-area: top`, and `position-try-fallbacks: flip-block` on the tooltip to tether it above the button and automatically flip below if there's not enough space

**Polyfill fallbacks** with proper feature detection for all three APIs:
- Interest Invokers (`interestfor`) — checks `interestForElement` on `HTMLButtonElement.prototype`
- Popover API — checks `popover` on `HTMLElement.prototype`
- CSS Anchor Positioning — checks `positionAnchor` in `document.documentElement.style`

The polyfills are downloaded locally in `polyfills/` rather than loaded from a CDN at runtime.
