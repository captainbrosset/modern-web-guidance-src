The landing page is complete. Here's a summary of what was built in `index.html`:

**Page structure:**
- Fixed header with nav links (Menu, About, Contact)
- Full-viewport hero section with an Unsplash coffee photo, overlay, headline, and CTA button
- Menu section with 3 signature drink cards
- About section with shop interior photo and story text
- Contact section and footer

**Three key features implemented:**

1. **Non-modal popover on hover (Interest Invokers + Popover API)** -- Each menu card has a "Details" button with `interestfor` pointing to a `popover="hint"` element. Hovering the button triggers the popover showing ingredients. CSS Anchor Positioning tethers each popover above its trigger button. Conditional polyfills are loaded for `interestfor`, `popover`, and `anchor-positioning` when native support is missing.

2. **Scroll-driven fade-in + scale (CSS `animation-timeline: view()`)** -- The hero image wrapper starts at `opacity: 0` and `scale(0.92)`, animating to full visibility as it enters the viewport. Wrapped in `@supports (animation-timeline: view())` so unsupported browsers simply show the image normally. A `prefers-reduced-motion` media query disables the animation for users who prefer it.

3. **Adaptive Loading (`loading-placeholder` attribute)** -- Both the hero image and the about section image use `loading-placeholder` with a tiny 20px-wide version of the same Unsplash photo. On slow networks (3G or worse), the browser serves the lightweight placeholder instead. A polyfill is conditionally loaded for browsers without native support.
