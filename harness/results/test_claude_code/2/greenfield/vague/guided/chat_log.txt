The landing page is built at `index.html`. Here's what's included:

### Features implemented

**1. Ingredients tooltip on "Details" button hover**
- Each menu card has a "Details" `<button>` using the `interestfor` attribute to declaratively trigger a `popover="hint"` tooltip on hover
- Tooltips are positioned with CSS Anchor Positioning (`position-anchor`, `position-area: top`, `position-try-fallbacks: flip-block`)
- Each card gets its own anchor name via CSS custom properties to avoid collisions
- Polyfills loaded conditionally for `interestfor`, Popover API, and CSS Anchor Positioning in unsupported browsers

**2. Hero image fade-in + scale on scroll**
- Uses `animation-timeline: view()` with a `fade-scale-in` keyframe that goes from `opacity: 0; scale(0.85)` to `opacity: 1; scale(1)`
- Wrapped in `@supports (animation-timeline: view())` so the image stays visible in unsupported browsers
- Includes `@media (prefers-reduced-motion: reduce)` to disable animation for users who prefer it

**3. Low-quality placeholder on slow networks**
- Uses the `loading-placeholder` attribute on the hero `<img>`, pointing to a tiny 20px-wide version of the same Unsplash image
- On slow connections (3G or worse), the browser loads the placeholder first
- Polyfill loaded conditionally for browsers without native support

**4. Page structure**
- Fixed navbar, full-viewport hero section, 3-card menu grid, about section, and footer
- Responsive layout using `clamp()`, `min()`, and `auto-fit` grid
- Unsplash stock photo for the hero image
