The landing page is built in `index.html`. Here's a summary of what's included:

**Page structure** — "Ember & Brew" coffee shop with a fixed nav bar, hero section, three-card menu grid, about/story section, and a footer.

**Three key features:**

1. **Details button with ingredients tooltip** — The "Details" button in the hero section reveals a styled tooltip on hover listing the signature blend ingredients (Ethiopian Yirgacheffe, Colombian Huila, etc.). It uses pure CSS `visibility`/`opacity` transitions with a downward-pointing arrow, and also activates on `:focus-within` for keyboard accessibility via `aria-describedby`.

2. **Scroll-driven hero image animation** — The hero image starts at `opacity: 0` and `scale(0.85)`. A passive `scroll` event listener interpolates both values linearly to `opacity: 1` / `scale(1)` as the user scrolls through the first 50% of the viewport height. It also runs on page load to handle cases where the page starts partially scrolled.

3. **Low-quality image placeholder (LQIP)** — On page load, the script checks the `navigator.connection` API (`effectiveType` and `saveData`). On slow networks (`slow-2g`, `2g`, `3g`, or data-saver mode), it loads a tiny 40px-wide version of the hero image with a CSS `blur(20px)` filter, then progressively swaps in the full 1200px image with a blur-to-sharp transition. On fast networks, it loads the high-quality image directly.

All images are sourced from Unsplash, and everything is self-contained in a single HTML file with no external dependencies.
