The landing page is ready at `index.html`. Here's what was built:

### Features

**1. Ingredients tooltip on "Details" buttons**
- Each menu card has a "Details" button that shows an ingredients tooltip on hover
- Uses the Popover API with `popover="hint"` and the `interestfor` attribute (Interest Invokers) for declarative hover-triggered popovers
- CSS Anchor Positioning tethers each tooltip to its button (each card gets a unique anchor name to avoid conflicts)
- Falls back to a CSS-only `:hover` tooltip for browsers without anchor positioning support
- Polyfills are conditionally loaded for Interest Invokers, Popover, and Anchor Positioning

**2. Scroll-driven hero image animation**
- The hero image fades in (opacity 0.3 to 1) and scales up (1x to 1.08x) as you scroll down
- Uses `animation-timeline: view()` with `animation-range: entry 0% entry 80%`
- Wrapped in `@supports (animation-timeline: view())` so unsupported browsers just see the image normally
- Respects `prefers-reduced-motion: reduce` by disabling the animation

**3. Adaptive loading for slow networks**
- The hero image uses the `loading-placeholder` attribute pointing to a tiny 20px-wide version of the same Unsplash image (`?w=20&q=10`)
- On constrained connections (3G or slower), the browser loads the placeholder first
- Polyfill is conditionally loaded for browsers without native support
- The about-section image uses standard `loading="lazy"` since it's below the fold

**4. General layout**
- Header with nav links, full-viewport hero section, about section, 4-card menu grid, and a footer
- Images are Unsplash stock photos (coffee cup hero, barista latte art)
- Warm brown/cream color scheme fitting a coffee shop brand
