The landing page is built. Here's a summary of what was created in `index.html`:

**Page structure:**
- Fixed navigation bar with logo and anchor links
- Hero section with headline, tagline, and a full-width coffee image from Unsplash
- Menu section with three drink cards in a responsive grid
- About section with the shop's story
- Footer

**Three key features implemented:**

1. **Hover-triggered popover (Interest Invokers + Popover API)** -- The "Details" button on the House Blend Latte card uses `interestfor="ingredients-popover"` to declaratively show a `popover="hint"` on hover. CSS Anchor Positioning (`position-anchor`, `position-area: top`, `position-try-fallbacks: flip-block`) tethers the popover above the button. Polyfills for Interest Invokers, Popover, and Anchor Positioning are conditionally loaded via feature detection.

2. **Scroll-driven animation (fade + scale)** -- The hero image uses `animation-timeline: view()` with a `hero-reveal` keyframe that transitions from `opacity: 0; scale(0.85)` to `opacity: 1; scale(1)` as the image enters the viewport. Wrapped in `@supports (animation-timeline: view())` so unsupported browsers just show the image normally. A `prefers-reduced-motion: reduce` media query disables the animation for accessibility.

3. **Adaptive Loading** -- The hero `<img>` uses `loading-placeholder` pointing to a tiny 20px-wide version of the same Unsplash image, so users on slow networks (3G or worse) get a lightweight placeholder instead of the full 900px image. A polyfill is conditionally loaded when the native API isn't available.
