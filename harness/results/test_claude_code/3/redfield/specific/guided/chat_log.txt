Here's a summary of the changes made to `index.html`:

**Removed:**
- The wrapping `.container` div (no longer needed — anchor positioning handles tethering)
- All JavaScript event listeners (`mouseover`/`mouseout`) and manual DOM manipulation (`tooltip.style.display`)
- The old `position: absolute` / `transform` based positioning on the tooltip
- `display: none` — the popover API handles visibility

**Added:**
- **`interestfor="tooltip"`** on the button — declaratively links the trigger to the popover on hover/focus (replacing the JS event listeners)
- **`popover="hint"`** on the tooltip div — uses the Popover API's `hint` type, purpose-built for tooltips
- **CSS Anchor Positioning** — `anchor-name: --tooltip-trigger` on the button and `position-anchor` / `position-area: top` on the tooltip, with `position-try-fallbacks: flip-block` so it flips below the button if there's not enough space above
- `border: none` on the tooltip to reset the default popover border
- **Three conditional polyfills** (loaded only when the browser lacks native support):
  - `interestfor` polyfill for Interest Invokers
  - `@oddbird/popover-polyfill` for the Popover API
  - `@oddbird/css-anchor-positioning` for CSS Anchor Positioning
