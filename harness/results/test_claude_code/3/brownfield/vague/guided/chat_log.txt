Here's a summary of what was done:

**Instant navigation via Speculation Rules API**

Every page on the site now includes a `<script type="speculationrules">` block that tells the browser to **prerender** same-origin pages at `"moderate"` eagerness. This means when a user hovers over or touches a link, the browser begins fully rendering that page in the background. By the time they click, the page is already loaded and the navigation appears instant. The `/logout` URL is excluded from prerendering since it triggers a state change.

**Pages created (13 total)**

All links from `index.html` now resolve to real pages:
- `/menu` - main menu with drinks and food
- `/menu/seasonal`, `/menu/espresso`, `/menu/cold-brew` - detail pages
- `/locations` - store locations and hours
- `/rewards` - loyalty program tiers
- `/account` - user profile and order history
- `/order-online` - location picker and popular items
- `/about` - company story
- `/careers` - open positions
- `/contact` - contact form
- `/privacy-policy` - privacy policy
- `/logout` - logged-out confirmation

**Other changes**
- Extracted shared CSS into `/styles.css` to avoid duplicating the full stylesheet across every page (also reduces bytes for prerendered pages since CSS is cached once)
- Made the logo a link back to home on all pages
- Used stock photos from Unsplash for imagery
