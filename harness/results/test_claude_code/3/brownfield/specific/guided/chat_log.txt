Here's what I added to `index.html` (just before `</head>`):

```html
<script type="speculationrules">
  {
    "prerender": [
      {
        "source": "document",
        "where": {
          "and": [
            { "href_matches": "/*" },
            { "not": { "href_matches": "/logout" } }
          ]
        },
        "eagerness": "moderate"
      }
    ]
  }
</script>
```

**How it works:**

- **Speculation Rules API** — a native web platform feature that tells the browser to speculatively pre-render pages the user is likely to navigate to, so they load near-instantly when clicked.
- **`"source": "document"`** — the browser automatically discovers matching links in the page (all nav links, footer links, etc. matching `/*`).
- **`"not": { "href_matches": "/logout" }`** — explicitly excludes `/logout` from pre-rendering, preventing accidental session termination or state changes.
- **`"eagerness": "moderate"`** — pre-renders on hover/pointer-down, balancing speed with resource usage (not as aggressive as `"eager"` or `"immediate"`).
- **Graceful degradation** — browsers that don't support the Speculation Rules API simply ignore the `<script type="speculationrules">` tag and navigation works normally.
