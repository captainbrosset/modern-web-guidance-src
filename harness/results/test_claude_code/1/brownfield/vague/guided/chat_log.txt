Here's a summary of everything that was done:

**Instant navigation via the Speculation Rules API**

Every page now includes a `<script type="speculationrules">` block that tells the browser to **prerender** same-origin pages at `"moderate"` eagerness — meaning when a user hovers over or starts to click a link, the browser speculatively loads and renders the target page in the background. By the time the click completes, the page is already fully rendered and the navigation appears instant. The `/logout/` path is excluded from prerendering since it triggers a state change.

This is a progressive enhancement — browsers that don't support speculation rules (Firefox, Safari as of now) simply ignore the script tag and navigate normally.

**Pages created**

All links in the original `index.html` now resolve to real pages:

**Other changes**

- Extracted shared CSS into `/styles.css` (linked from all pages) to keep page payloads small and enable browser caching — the stylesheet loads once and is reused across all navigations
- All links use trailing-slash directory format (`/menu/` → `menu/index.html`) for clean URLs
- Images use Unsplash stock photos
