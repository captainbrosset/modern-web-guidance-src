**Instant navigation system** (`instant-nav.js`):
- **SPA-style navigation**: Clicking internal links fetches the target page via `fetch()`, then swaps only the `#page-content` div's innerHTML and updates `document.title` and `history.pushState` -- no full page reload
- **Prefetch on hover/touch**: When a user hovers over or touches a link, the target page is fetched in the background and cached in memory, so by the time they click, the content is already available
- **Prefetch on viewport intersection**: Uses `IntersectionObserver` with a 200px margin to prefetch any link that scrolls into view
- **Eager prefetch for nav links**: All navigation bar links are prefetched immediately on page load since they're always visible
- **Browser history support**: Back/forward buttons work via `popstate` handler
- **Graceful fallback**: If fetching fails or the target page doesn't have the expected `#page-content` structure, it falls back to normal full-page navigation
- **`/logout` excluded**: The logout link is excluded from SPA navigation so it can perform a proper server-side logout

**Shared CSS** (`styles.css`):
- Extracted from inline styles into a shared file used by all pages
- Added a quick 120ms opacity transition on `#page-content` for smooth visual feedback during navigation
