The landing page is built at `index.html`. Here's what's included:

### Features implemented

**1. Ingredients tooltip on the "Details" button**
- Uses the Popover API (`popover="hint"`) with `interestfor` attribute for hover-triggered display
- Positioned with CSS Anchor Positioning, tethered above the button with `flip-block` fallback
- Shows a list of signature blend ingredients (Ethiopian Yirgacheffe, Colombian Supremo, cocoa, hazelnut, brown sugar)
- Polyfills loaded conditionally for Interest Invokers, Popover, and Anchor Positioning in unsupported browsers

**2. Hero image fade-in + scale on scroll**
- Uses `animation-timeline: view()` (scroll-driven animations) to fade the image from `opacity: 0` / `scale(0.85)` to fully visible / full size as it enters the viewport
- Wrapped in `@supports (animation-timeline: view())` so unsupported browsers simply see the image at full opacity
- Respects `prefers-reduced-motion: reduce` by disabling the animation

**3. Low-quality placeholder on slow networks**
- Uses the `loading-placeholder` attribute on the hero `<img>`, pointing to a tiny 80px-wide, quality-10 version of the same Unsplash photo
- On constrained connections (3G or slower), the browser loads the placeholder first
- Polyfill loaded conditionally for browsers without native support

**4. General page structure**
- Fixed navigation bar with blur backdrop
- Hero section with headline, description, and CTA buttons
- Menu cards section with 3 featured drinks (lazy-loaded images via `loading="lazy"`)
- About section with shop story
- Footer with address and contact info
- All images sourced from Unsplash stock photos
