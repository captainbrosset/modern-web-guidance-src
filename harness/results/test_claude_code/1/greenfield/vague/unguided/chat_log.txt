The landing page is ready at `index.html`. Here's what was built:

**Page structure** — "Ember & Brew" coffee shop with a fixed navbar, hero section, features grid, menu highlights, and footer. Uses an Unsplash stock photo for the hero image.

**Three key features:**

1. **Details button with ingredients tooltip** — The "Details" button in the hero section shows a styled tooltip on hover listing the signature blend ingredients (Ethiopian beans, blueberry/chocolate notes, milk options, sugar, vanilla syrup). Uses pure CSS `:hover` / `:focus-within` for the reveal, plus `aria-describedby` and `role="tooltip"` for accessibility.

2. **Scroll-driven hero image animation** — The hero image starts at `opacity: 0` and `scale(0.85)`. A `scroll` event listener calculates a 0→1 progress value based on the image's position in the viewport, interpolating opacity to 1 and scale to 1.0 as you scroll down.

3. **Low-quality image placeholder (LQIP)** — An inline SVG placeholder (brown/tan shapes suggesting a coffee cup) is displayed immediately with a 20px blur. The script checks `navigator.connection.effectiveType` and `saveData` — on slow networks (`slow-2g`, `2g`, `3g`) it swaps the `src` to a smaller 400px/q60 variant and keeps the placeholder visible until load completes. On fast networks the placeholder fades out as soon as the full 800px image is ready.
