The landing page is built. Here's a summary of what's in `index.html`:

**Layout & Design**
- Fixed navigation bar with blur backdrop
- Full-viewport hero section with headline, tagline, and hero image (Unsplash stock photo)
- 3-column menu grid with signature drink cards
- Dark-themed "About" section and footer

**Feature 1: Hover Popover (Interest Invokers + Popover API)**
- The "Details" button on the Ember Espresso card uses `interestfor="ingredients-popover"` to declaratively trigger a non-modal `popover="hint"` on hover
- The popover is anchored to the button using CSS Anchor Positioning (`position-anchor`, `position-area: top`, `position-try-fallbacks: flip-block`)
- Polyfills loaded conditionally for Interest Invokers, Popover API, and CSS Anchor Positioning

**Feature 2: Scroll-driven Animation**
- The hero image wrapper uses `animation-timeline: view()` with a `hero-reveal` keyframe that fades in (`opacity: 0 -> 1`) and scales up (`scale(0.92) -> scale(1)`) as the element enters the viewport
- Wrapped in `@supports (animation-timeline: view())` so unsupported browsers see the image normally
- Respects `prefers-reduced-motion: reduce`

**Feature 3: Adaptive Loading**
- The hero `<img>` uses `loading-placeholder` pointing to a tiny 20px-wide version of the same Unsplash image, which gets served on slow networks (3G or worse)
- Polyfill loaded conditionally via `'loadingPlaceholder' in HTMLImageElement.prototype` check
