The landing page is complete. Here's a summary of the three key features implemented:

**1. Non-modal popover on hover (Popover API)**
- The "Details" button uses the native HTML `popover` attribute. JavaScript listens for `pointerenter`/`pointerleave` on both the button and the popover itself, calling `showPopover()` and `hidePopover()`. A small delay on hide lets users move their cursor into the popover without it closing. The popover is positioned near the button via the `toggle` event and `getBoundingClientRect()`.

**2. Scroll-driven animation (CSS `animation-timeline: view()`)**
- The `.hero-img-wrapper` starts at `opacity: 0` and `scale: .85`, and animates to full opacity and scale using a `@keyframes heroReveal` animation bound to `animation-timeline: view()` with `animation-range: entry 0% entry 60%`. A `@supports not (animation-timeline: view())` fallback ensures the image is visible in browsers that don't yet support this API.

**3. Adaptive Loading via Network Information API (`navigator.connection`)**
- On page load, the script checks `navigator.connection.effectiveType` and `saveData`. On slow connections (`slow-2g`, `2g`, `3g`, or data-saver on), it loads a tiny 40px-wide thumbnail and keeps a semi-transparent placeholder with a "Low-res preview" message. On fast connections, it loads the full 1200px image and hides the placeholder on load. A `change` listener upgrades the image if the connection improves.

All images are sourced from Unsplash. The page uses no external dependencies — everything is native HTML, CSS, and JavaScript.
