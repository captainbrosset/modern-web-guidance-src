# The DevTools Times

This project is a sample news application built with Astro, designed to serve as a testing ground for new features in Chrome DevTools.


```sh
pnpm install && pnpm run start
```

The application will now be running on `http://localhost:11224`.

## Tech Stack

- **[Astro](https://astro.build/)**: The web framework for building content-driven websites.
- **[React](https://react.dev/)**: Used for UI components.
- **[Tailwind CSS](https://tailwindcss.com/)**: For styling the application.
- **[Astro DB](https://docs.astro.build/en/guides/database/)**: Powers the comments feature, using a libSQL-compatible database.
