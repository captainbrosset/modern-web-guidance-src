# Accessibility Guides

This directory contains guides for creating inclusive web applications using modern accessibility features and patterns, ensuring products are usable by everyone, regardless of their browsing context or ability.

## Available Guides

| Guide Name | Description | Web Feature IDs |
| :--- | :--- | :--- |
| [`accessible-error-announcement`](./accessible-error-announcement) | Synchronize programmatic accessibility states (like aria-invalid) with the visual :user-invalid state to ensure screen reader users receive error feedback only after interaction, mirroring the visual experience. | `user-pseudos` |

## How to use

1.  **Identify the problem**: Determine which accessibility enhancement is needed based on the descriptions above.
2.  **Navigate to the guide**: Click on the guide name link or navigate to the subdirectory.
3.  **Read `guide.md`**: Each guide contains detailed implementation steps, example code, and best practices.
